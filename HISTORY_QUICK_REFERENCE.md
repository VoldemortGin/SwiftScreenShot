# 截图历史 - 快速参考

## 快捷键

| 操作 | 快捷键 |
|------|--------|
| 打开历史窗口 | ⌘H |
| 截图 | ⌃⌘A |

## 历史窗口操作

### 缩略图悬停菜单
- 👁️ **预览**：查看完整图片
- 📋 **复制**：复制到剪贴板
- ✏️ **编辑**：在编辑器中打开
- 📌 **固定**：标记为重要（不会被自动清理）
- 🗑️ **删除**：删除这张截图

### 工具栏功能
- **搜索框**：按格式筛选（输入 png、jpg 等）
- **日期筛选**：
  - 全部
  - 今天
  - 昨天
  - 最近7天
  - 最近30天
- **清空历史**：
  - 仅删除未固定
  - 全部删除

## 设置选项

### 历史记录部分
1. **自动保存到历史**
   - 默认：开启
   - 关闭后不再自动保存新截图

2. **历史数量上限**
   - 选项：10 / 20 / 50 张
   - 默认：20 张
   - 注意：固定的截图不受限制

3. **存储位置**
   - 默认：`~/Library/Application Support/SwiftScreenShot/History`
   - 可自定义路径
   - 点击"重置"恢复默认

## 固定功能

### 如何固定截图
1. 在历史窗口找到截图
2. 悬停显示操作菜单
3. 点击📌图标

### 固定的好处
- ✅ 不会被自动清理（即使超过数量上限）
- ✅ 有橙色图钉标记
- ✅ 清空历史时可选择保留

## 存储结构

```
~/Library/Application Support/SwiftScreenShot/History/
├── Images/                                   # 完整截图
│   ├── screenshot_20231214_153020_123.png
│   └── ...
├── Thumbnails/                               # 缩略图
│   ├── thumb_20231214_153020_123.png
│   └── ...
└── index.json                                # 历史索引
```

## 常见场景

### 场景1: 找回最近的截图
1. 按 ⌘H 打开历史
2. 找到需要的截图
3. 点击复制或预览

### 场景2: 重新编辑旧截图
1. 打开历史窗口
2. 悬停在截图上
3. 点击 ✏️ 编辑图标
4. 在编辑器中修改

### 场景3: 保留重要截图
1. 对重要截图点击 📌 固定
2. 它们不会被自动清理
3. 可以长期保存

### 场景4: 清理磁盘空间
1. 打开历史窗口
2. 点击"清空历史"
3. 选择"仅删除未固定"保留重要内容
4. 或选择"全部删除"彻底清空

### 场景5: 按日期查找
1. 打开历史窗口
2. 使用日期筛选器
3. 选择"今天"、"昨天"或其他选项

## 性能提示

- ✅ 缩略图仅 200x200，加载快速
- ✅ 完整图片延迟加载，节省内存
- ✅ 建议定期清理历史释放空间
- ✅ 固定功能仅用于真正重要的截图

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 历史不显示 | 检查设置中"自动保存到历史"是否开启 |
| 缩略图不显示 | 删除 Thumbnails 文件夹后会自动重建 |
| 找不到历史数据 | 检查是否更改了存储位置 |
| 占用空间太大 | 降低数量上限或定期清空 |

## 技巧

1. **快速访问**：将历史窗口固定在一侧，随时查看
2. **格式搜索**：输入 "png" 快速找到所有 PNG 格式截图
3. **批量固定**：重要项目截图全部固定，项目结束后一次性清理未固定项
4. **备份历史**：定期复制 History 文件夹到其他位置备份

# 截图编辑工具 - 快速入门

## 快速上手

### 启用编辑功能

1. 打开 SwiftScreenShot 应用
2. 点击菜单栏图标，选择"设置..."
3. 勾选"截图后自动编辑"选项
4. 完成！现在截图后会自��打开编辑器

### 基础使用流程

```
截图 → 自动打开编辑器 → 添加标注 → 保存完成
```

## 编辑工具说明

### 1. 选择工具 👆
- **图标**: 手指
- **用途**: 选择和移动已有标注
- **操作**:
  - 点击标注选中
  - 拖拽移动位置
  - 按 Delete 删除

### 2. 箭头工具 ↗️
- **图标**: 箭头
- **用途**: 绘制指向性箭头标注
- **操作**: 按住鼠标拖拽绘制
- **可调**: 颜色、粗细

### 3. 文字工具 📝
- **图标**: T 字形
- **用途**: 添加文字说明
- **操作**: 点击位置 → 输入文字 → 确定
- **可调**: 颜色、字号（12-48）

### 4. 矩形工具 ▭
- **图标**: 矩形
- **用途**: 框选重点区域
- **操作**: 拖拽绘制矩形
- **特点**: 圆角边框 + 半透明填充
- **可调**: 颜色、粗细

### 5. 椭圆工具 ○
- **图标**: 圆形
- **用途**: 圈出关键内容
- **操作**: 拖拽绘制椭圆
- **特点**: 半透明填充
- **可调**: 颜色、粗细

### 6. 马赛克工具 ⬛
- **图标**: 方格
- **用途**: 隐藏敏感信息
- **操作**: 拖拽选择要模糊的区域
- **效果**: 自动像素化处理

## 快捷键速查

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 撤销 | `⌘Z` | 撤销上一步操作 |
| 重做 | `⌘⇧Z` | 重做下一步操作 |
| 保存 | `⌘S` | 保存并完成编辑 |
| 取消 | `ESC` | 取消编辑（需确认） |

## 使用技巧

### 💡 颜色选择
- 点击颜色块打开系统颜色选择器
- 常用颜色：红色（警示）、黄色（高亮）、蓝色（说明）
- 建议使用对比度高的颜色

### 💡 粗细调整
- **细线 (1-2)**: 适合精确标注
- **中等 (3-5)**: 适合一般使用
- **粗线 (6-10)**: 适合强调重点

### 💡 文字标注
- 字号 12-16: 小说明
- 字号 18-24: 正常标注
- 字号 28-48: 大标题

### 💡 马赛克使用
- 用于隐藏个人信息（姓名、电话、地址）
- 用于保护隐私内容（密码、账号）
- 建议覆盖范围稍大一些

## 常见问题

### Q: 如何取消编辑？
**A**: 按 ESC 键，或点击"取消"按钮。会提示确认以防误操作。

### Q: 编辑完成后图片保存在哪里？
**A**:
- 总是复制到剪贴板
- 如果启用了"同时保存到文件"，会保存到设置的路径
- 文件名格式：`Screenshot_20231214_153045.png`

### Q: 可以编辑已有的标注吗？
**A**:
- 可以移动标注位置（选择工具）
- 可以删除标注（选中后按 Delete）
- 暂不支持修改标注内容（如改文字）

### Q: 误操作怎么办？
**A**: 使用撤销功能（⌘Z），支持最多50步撤销。

### Q: 如何关闭自动编辑？
**A**: 设置 → 取消勾选"截图后自动编辑"。

### Q: 标注会影响图片质量吗？
**A**: 不会。标注使用矢量绘制，最后渲染到图片时保持原始截图质量。

## 典型使用场景

### 📋 Bug 报告
1. 截取错误界面
2. 用箭头指出问题位置
3. 用文字说明错误现象
4. 用矩形框出相关区域

### 📖 教程制作
1. 截取操作界面
2. 用箭头标注操作步骤
3. 用文字添加说明
4. 用椭圆突出重点按钮

### 🔒 隐私保护
1. 截取包含敏感信息的内容
2. 用马赛克遮盖个人信息
3. 确保信息完全不可见
4. 保存并分享

### ✏️ 反馈标注
1. 截取设计稿或文档
2. 用矩形/椭圆标记问题区域
3. 用文字添加修改建议
4. 用箭头指明具体位置

## 最佳实践

✅ **推荐做法**:
- 使用对比色标注（如红色箭头 + 白色背景）
- 文字保持简洁（5-10字）
- 标注不要过多（3-5个为宜）
- 重要信息用多种方式标注（箭头 + 文字）

❌ **避免**:
- 标注颜色与背景相近
- 文字过长导致难以阅读
- 标注过密影响观看
- 马赛克范围过小导致信息泄露
